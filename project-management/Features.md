## Fonctions du projet

### Parties

- Noter les scores pour une session de jeu en cours
- Enregistrer les résultats d'une partie

### Feuilles de score

- Créer un modèle de feuille de score personnalisé
- Définir un modèle de feuille de score par défaut
- Enregistrer le score d'une session
- Gestion de règles de jeux différents (différents modèles de score)
- Partager des modèles communautaires
  - Upvotes

### Jeux

- Liste de jeux
- Règles des jeux
- Succès/badges
- Statistiques jeu

### Compte utilisateur

- Historique de parties
  - Enregistrement de score final
  - Statistiques joueur
- Bibliothèque personnelle
- > Tags sur les jeux (type "veut acheter", "possède", "a joué")
- Tables de jeux / amis / guildes / taverne

### Utilitaires

- Dés
  - Pièce pile ou face
- Minuteur
- Random wheel
- Sélection de premier joueur
  - Flèche
  - Age
  - Autres
- Création de presets d'outils (comme des modules, associés à un jeu)

### Guide utilisateur

- Création de modèle de feuille de score
